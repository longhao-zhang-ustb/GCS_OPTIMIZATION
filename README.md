# GCS_OPTIMIZATION
This is a project utilising machine learning to enhance defect detection in GCS. Please download the code once you have obtained data access permissions.
+ Project entry file: main.py
+ Python dependency libraries：requirements.txt
+ Training and test files： train_zigong.py & evaluate_zigong.py
+ shap analysis file： exp3_1_all_metrics_shap_analysis.py & exp3_2_sample_shap_analysis.py
+ Data Set Application Address：https://www.physionet.org/content/icu-infection-zigong-fourth/1.1/
+ Should you have any further queries, please do not hesitate to contact me.
